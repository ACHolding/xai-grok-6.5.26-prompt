## Overview

CatSDK is the official AI agent framework for **vibecoding** — the chaotic, high-speed, single-file, 60FPS Python-powered development style pioneered by Justin L Liao of AC Holdings.

Built for seamless integration with Grok, CatSDK turns vibe sessions into production-grade prototypes, emulators, tools, and agents at lightspeed.

## Core Philosophy

- **Vibe First**: Code what feels right. Let the flow guide you.
- **Single File Maximalism**: One .py file to rule them all.
- **60FPS Everything**: If it doesn't run buttery smooth, it gets refactored.
- **Cat Energy**: Maximum chaos, maximum fun, zero unnecessary boilerplate.
- **Personalization AI**: Everything adapts to the user in real time.
